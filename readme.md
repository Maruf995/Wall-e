# ПО для робота "Wall-e" 

### Запустить проект локально
1. Скопировать проект
```python
git clone https://github.com/Maruf995/Wall-e.git
```

2. Загрузить библиотеки:
```
pip install -r requirements.txt
```

3. Запустить проект:
```
python main.py
```

4. Запустить проект (Без ИИ):
```
python wht_ai.py
```
